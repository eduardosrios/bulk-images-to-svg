# Code of Conduct

## Our Pledge

We want Bulk Image to SVG to be a welcoming project for contributors, users,
and maintainers. Everyone participating in this project is expected to treat
others with respect and patience.

## Expected Behavior

- Be respectful when discussing bugs, design decisions, and tradeoffs.
- Keep feedback specific, technical, and actionable.
- Assume good intent when reading comments or issue reports.
- Help keep conversations focused on improving the project.

## Unacceptable Behavior

- Harassment, insults, intimidation, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Repeated off-topic or disruptive comments.

## Reporting

If you see behavior that violates this code of conduct, please open a private
conversation with the repository owner or use GitHub's reporting tools.

Reports will be reviewed with care and handled as privately as possible.
